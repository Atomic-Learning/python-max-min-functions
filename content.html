The built-in function max() and min() returns the largest and smallest of the arguments passed to them.

max() and min() on Scalars

>

When passed a set of individual numerical arguments, max() returns the largest and min() returns the smallest.

print(max(1, -2, 3)) print(min(1, -2, 3))

max() and min() on Iterables

When passed an iterable (such as a list) of numerical values , max() returns the largest item and min() returns the smallest item.

values = [1, 2, -3, 4] print(max(values)) print(min(values))
